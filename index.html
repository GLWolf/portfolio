Green Lantern Quiz

Links

  • Live App
  • Repository

Description

This app will test your knowledge of the green lantern comics. It’s a 10 question quiz that uses HTML, CSS, JavaScript, and JQuery. This is a fun app to play with and test your knowledge. I plan on updating a few futures and adding audio files to the quiz, which will read the questions and answers to you also give a brief history for each question.

Hurdles

Most of the hurdles I face during this project were typos and syntax errors. Coding at 3 am is not the best idea. Another challenge I faced was getting the styling for the text and the background to work together. I have already started on the audio function and it has been challenging and fun.

What I learned

I learned a lot during my time building this app. Creating listeners that would iterate to the correct question and bring up the correct answers. I am learning to add audio files and will add them to the app. Also being careful about what I type is another important lesson I learned from this app.

Game Sales

Links

  • Live App
  • Repository

Description

This app will helps find sales for pc games. It will give you the normal price, the sale price, a link to Metacritic, and the website were the sales at. If you search for a game that you know is a pc game and it does not show up it’s because the game is not on sale.

Hurdles

One challenge I faced during this project was that API website only works on Internet Explorer and it was not clear how to use the information. A lot of trial and error. Another issue I had was with styling the image that appeared for the game. Most images were not the same size with a few being tiny and others being huge.

What I learned

I learned a lot about problem solving, especially with working on two different APIs and having them work cohesively together. I also got better grasp of creative styling. For example, if you’re on the landing page and hover over the links that show under the store list of sale, a small image of the game will appear.

Work Out With Friends

Links

  • Live App
  • Client repo
  • Server repo

Description

WorkOutWF is an app that help to connect with others with a similar workout schedule. You can create a profile add in a day and general time that you are available to work out. After you can search by day and available times and find other users with that availability.

Hurdles

One big obstacle that I overcame on this project was how to get the user to search for other user by the date and time. A user updating their profile was easy, but grabbing the data was difficult. Since I only wanted some of the data, not the user’s entire list. I created a dropdown search so the user picks from a list search occurs it only grabs the requested data.

What I learned

I learned that it is better work on the basics of the app first, and not to get lost in all the add-ons features that you want to add.

What That Data Do

Links

  • Live App
  • Client repo
  • Server repo

Description

WTDD enables users to create visual representations of data from their csv files. Users will be able to graph x y coordinates, and representing them visually as bar, line, and scatter plot graphs. Make it easy for students/non-technical people to make graphs and charts.

Hurdles

Working on a team was a great experience, but something I had to learn quickly was to learn how others like to wright code. One amazing thing about coding is that there’s many ways to tackle a problem and create a solution. So I had to learn how each member on my team wrote their code so if I had to fix a bug or add a functionality I was not lost on where to start.

What I learned

Working with github and creating multiple branches pushing, pulling, and merging was a headache at first. It took some trial and error to understand how to successfully merge branches with only a few conflicts. I truly loved working with a team and learning from each member.
